# Source Note

This project is based on indexed person-level records from Anton Chekhov’s 1890 Sakhalin census.

## Unit of analysis

Individual person record.

## Geographic scope

Sakhalin Island.

## Temporal scope

1890.

## Source format

Indexed PDF book.

## Data access note

The original PDF and full person-level records are not included in this repository until copyright, access, and ethical considerations are reviewed.
